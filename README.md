# Simple 3D Rendering Engine (C / OpenGL)

This project is a lightweight 3D rendering engine written in C using OpenGL.  
It was built as a personal project to explore the fundamentals of real-time graphics and the design of performance-oriented rendering pipelines.

The engine focuses on implementing core components of a modern graphics pipeline while keeping the codebase small and easy to understand.

## Features

- OpenGL-based rendering pipeline
- Basic 3D scene rendering
- Object transformations (model / view / projection)
- Shader-based rendering
- Camera system for navigating 3D scenes
- Modular code structure for experimenting with graphics concepts

## Technologies

- C
- OpenGL
- GLSL shaders

The implementation is inspired by the excellent resource  
**LearnOpenGL – Graphics Programming** by Joey de Vries:  
https://learnopengl.com

## Build

Currently the project is built using a simple build script.
